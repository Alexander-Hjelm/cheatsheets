# Help

## help

Prints the PowerShell help section.

```
help
```

## gcm

Searches for a specific command or a substring of a command.

```
gcm Copy-Item
```

Or

```
gcm *Copy-*
```

# Connectivity

## curl

```
curl -v www.example.com
```

## Test-Connection

```
Test-Connection -TargetName www.example.com
 ```
 
### Parameters:
 
- **-Traceroute**
- **-TcpPort <int>**
