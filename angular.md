## Getting started

Create a new angular application:

```bsah
sudo npm i -g @angular/cli
nvm install --lts
nvm use --lts
ng new my-angular-app
cd my-angular-app
npm run start
```

## Generate component

```bsah
ng generate component components/header
```

## Install packages with ng (font-awesome)

```bash
ng add @fortawesome/angular-fontawesome@<version>
```
